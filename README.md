# 🃏 Blackjack

## プロジェクト概要
HTML/CSS/JavaScriptのみで実装したシングルファイル構成のブラックジャックWebアプリ。
スマホ・PCどちらでも快適にプレイできるレスポンシブデザイン。

## 🎮 機能一覧
- カードの配布・ヒット・スタンド
- 勝敗判定（バスト・ブラックジャック・通常勝敗・プッシュ）
- チップ賭け（$10 / $50 / $100 / $500）
- 残高管理（初期$1,000・ペイアウト・ブラックジャック1.5倍）
- カードアニメーション（配布・フリップ）
- レスポンシブ対応（スマホ320px〜PC1200px+）

## 🕹️ 使い方（プレイ手順）
1. チップボタン（$10〜$500）をクリックしてベット額を設定
2. **Deal** ボタンでゲーム開始（カードが2枚ずつ配られる）
3. **Hit** でカードを追加 / **Stand** でターン終了
4. ディーラーが自動でカードを引き、勝敗が決定
5. 結果確認後、**Next Game** で次のラウンドへ

| 項目 | 内容 |

| HTML | セマンティックなUI構造 |
